# 설치 방법:

## 전제조건

1. Node JS (LTS) — [ 다운로드 ] ( 튜토리얼에서 v10.15.3 LTS 사용 )
2. Google OTP — [ 다운로드 : Android] [ 다운로드 : iOS]

```
-- back-end 폴더로 이동
nvm install 10.15.3
nvm use 10.15.3

npm install

node app.js

```

```
-- front-end 폴더로 이동

npm intall @angular/cli@7.3.8

npm install

node start

```
